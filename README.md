# Sewa AC Cikarang

Solusi Sewa AC Bulanan untuk Rumah, Kantor, Gudang & Proyek.

This repository contains the official, high-performance static landing page for **Sewa AC Cikarang**. Designed with a minimal, modern tech-inspired layout, it features a custom Daikin Blue styling, fully responsive elements, and optimized loading properties.

## Tech Stack
- **Structure**: HTML5
- **Style**: Custom CSS3 (Daikin Blue color system, responsive grid layouts, and custom resets)
- **Logic**: Vanilla JavaScript (< 100 lines) for interactive components
